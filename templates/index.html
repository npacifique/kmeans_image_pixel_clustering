This project aims to use the K-means algorithm to create clusters of dominant pixels(s) from the uploaded image. K-means is an unsupervised machine learning algorithm that creates clusters of similar data points by bringing similar points (pixels in the current context) closer to their mean, also known as the centroid. The user is required to provide the number of clusters (colors to extract)
